# AOD Landing Page

A landing page for a bicycle shop AOD. Portfolio project.

## Installation

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

### Production build

```bash
npm run build
```

## Technologies used

- HTML5
- CSS, SASS
- JavaScript

## Features

- Webpack module bundler
- Responsive Web Design
- Sections designed with CSS Flexbox layout
- Slider
- Hamburger mobile navigation menu
